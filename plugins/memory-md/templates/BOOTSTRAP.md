# Memory Setup

Welcome to memory-md plugin. This is your first run.

## Setup Steps

1. Ask me (the AI) questions to gather information
2. I will create MEMORY.md, IDENTITY.md, and USER.md
3. After setup, delete this file with: `/memory-md delete bootstrap`

## Questions

### For MEMORY.md (Long-term Memory)

- What are your preferred tech stacks?
- Any coding conventions I should follow?
- Important project decisions to remember?

### For IDENTITY.md (AI Identity)

- What name should I use?
- Any persona preferences?
- Communication style preferences?

### For USER.md (User Profile)

- Your name
- Your role/title
- Timezone
- Preferred working hours

## After Setup

Your memory files will be stored in `~/.claude/memory/`
They will be automatically loaded at the start of every session.

Use `/memory-md` command to manage your memory files.
